# External References

This document contains links to external documentation referenced by Lattice.

## Git

- [git update-ref](https://git-scm.com/docs/git-update-ref) - Safely update refs
- [git rebase](https://git-scm.com/docs/git-rebase) - Reapply commits on top of another base
- [git push --force-with-lease](https://git-scm.com/docs/git-push#Documentation/git-push.txt---force-with-leaseltrefnamegt) - Safe force push

## GitHub

- [REST Pull Requests API](https://docs.github.com/en/rest/pulls/pulls) - Create and manage PRs
- [REST Reviewers API](https://docs.github.com/en/rest/pulls/review-requests) - Request PR reviews
- [REST Merge API](https://docs.github.com/en/rest/pulls/pulls#merge-a-pull-request) - Merge PRs
- [GraphQL API](https://docs.github.com/en/graphql) - For draft PR toggling and advanced operations
